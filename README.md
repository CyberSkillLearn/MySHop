**MySHop** is a luxury perfume e-commerce web application developed as a semester project for IICT. The platform provides a sleek, high-end user experience for browsing and purchasing exclusive fragrances for both men and women.

---

## üöÄ Project Overview
This project simulates a real-world shopping experience for perfumes. It features a modern 2D interface with a professional color palette and interactive elements built entirely with core web technologies.

### Core Objectives:
* Implement a responsive UI for different product categories.
* Manage a dynamic shopping cart system with real-time price updates.
* Simulate a secure checkout and user authentication flow.

## ‚ú® Key Features
* **Dynamic Product Catalog**: The Home page displays signature products with high-quality imagery and detailed descriptions.
* **Category Filtering**: Separate sections for "For Men" and "For Women" to streamline the shopping experience.
* **Top Sales Tracking**: A dedicated page for the highest-selling products, arranged in a clean grid layout.
* **Advanced Shopping Cart**: 
    * Add items from any page.
    * Real-time quantity adjustment (increase/decrease).
    * Subtotal and total price calculations automatically adjuste.
* **Payment Processing**: A simulation of a credit card gateway featuring fields for cardholder name, number, and CVV.
