## üöÄ Project Overview
This project simulates a real-world shopping experience for perfumes. It features a modern 2D interface with a professional color palette and interactive elements built entirely with core web technologies.

### Core Objectives:
* [cite_start]Implement a responsive UI for different product categories[cite: 190].
* [cite_start]Manage a dynamic shopping cart system with real-time price updates[cite: 207].
* [cite_start]Simulate a secure checkout and user authentication flow[cite: 208, 209].

## ‚ú® Key Features
* [cite_start]**Dynamic Product Catalog**: The Home page displays 7 signature products with high-quality imagery and detailed descriptions[cite: 225].
* [cite_start]**Category Filtering**: Separate sections for "For Men" and "For Women" to streamline the shopping experience[cite: 200, 201].
* [cite_start]**Top Sales Tracking**: A dedicated page for the highest-selling products, arranged in a clean grid layout[cite: 202, 227].
* **Advanced Shopping Cart**: 
    * [cite_start]Add items from any page[cite: 199].
    * [cite_start]Real-time quantity adjustment (increase/decrease)[cite: 207].
    * [cite_start]Subtotal and total price calculations automatically adjusted[cite: 207].
* [cite_start]**User Authentication**: Securely designed Login and Signup forms with input validation[cite: 228, 229].
* [cite_start]**Payment Processing**: A simulation of a credit card gateway featuring fields for cardholder name, number, and CVV[cite: 209, 231].
